**❌ GUI doesn't load after executing**
→ Detach your executor, re-attach it to the Roblox process, and execute again. Ensure you're using a supported executor version.

**❌ Spawned pets aren't showing as tradable**
→ This usually means the script version is outdated. Download the latest release — the adopt me script pet spawner tradable module is patched with every major game update.

**❌ Auto trade keeps sending failed offers**
→ Open the Trade Config panel and verify your pet value tiers are correctly mapped. Offers that don't match the game's current trade validation rules will be auto-rejected server-side.

**❌ Auto grind stops completing tasks**
→ Some seasonal tasks change structure after game events. Enable "Dynamic Task Detection" in the auto grind settings and re-execute.

**❌ Scam detection is blocking legitimate trades**
→ Adjust the sensitivity slider in the Scam Detection settings from "Strict" to "Moderate" to reduce false positives on unusual but valid offers.

**❌ All pets module only shows partial collection**
→ Confirm you're on the latest script version. The all pets registry is updated with every Adopt Me content patch. Older versions will be missing newly added pets.

**❌ Script crashes on mobile during auto farm**
→ Mobile devices with less than 4GB RAM may struggle with multiple modules active simultaneously. Disable ESP and Auto House Manager when running auto farm on lower-end Android devices.
